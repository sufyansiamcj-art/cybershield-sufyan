# 🛡️ CyberShield - Web Security Scanner

تطبيق ويب احترافي تم تطويره باستخدام **Python** و **Flask**، مصمم لفحص المواقع الإلكترونية، استخراج البيانات الأساسية، وتحليل أمان الروابط مع إمكانية استخراج التقارير بصيغة PDF.

## 🚀 المميزات (Features)
* **فحص الروابط:** التأكد من حالة الموقع (Status Code) وسرعة الاستجابة.
* **تحليل البيانات:** استخراج عناوين الـ IP، بيانات الـ SSL، وتفاصيل السيرفر.
* **تقارير احترافية:** توليد ملف PDF يحتوي على نتائج الفحص كاملة بضغطة زر.
* **واجهة عصرية:** تصميم متجاوب (Responsive) يدعم اللغة العربية والإنجليزية.

## 🛠️ الأدوات المستخدمة (Tech Stack)
* **Backend:** Python / Flask
* **Frontend:** HTML5, CSS3 (Bootstrap), JavaScript
* **Deployment:** Render / GitHub
* **Libraries:** Requests, BeautifulSoup4, ReportLab
